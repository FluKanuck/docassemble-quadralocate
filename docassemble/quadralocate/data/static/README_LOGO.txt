QUADRA LOGO PLACEHOLDER
=======================

Place your Quadra logo file here with the name: quadra_logo.png

Recommended specifications:
- Format: PNG with transparent background
- Size: 300x100 pixels (or similar aspect ratio)
- Resolution: 72-150 DPI for web display

The logo will be used in:
- Report headers
- PDF cover page
- Web interface branding

To add the logo:
1. Export your logo as PNG
2. Name it "quadra_logo.png"
3. Place it in this folder (data/static/)
4. Update the interview YAML if needed to reference it
